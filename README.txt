All files are run with the following format from terminal:
python3 [filename] < infile.txt
Where [filename] is one of my .py files and infile.txt is a text file with the same format as the infile.txt file from this repository

Sources:
The AI:MA code repository was used for the defining the problem superclass, nodes, Astar search algorithms, hill climbing algorithm, and simulated annealing algorithm.

The website geeksforgeeks.org was used for the minimum spanning tree algorithm for an adjacency matrix.
